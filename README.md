# React + TypeScript + Vite

## Additions to the basic vite + react + typescript template

- Basic empty project template for React with TypeScript using Vite as the build tool.
- Features added to the base template:
  - Tailwind CSS for utility-first styling
  - `clsx` and `tailwind-merge` for conditional and conflict-free class name management
  - Tailwind-Animated for pre-defined animations

## Setup

1. Ensure you have Node.js installed on your machine.
2. Clone the repository or create a new Vite project with React and TypeScript.
3. Install the necessary dependencies:

```bash
npm install
```
4. Start the development server:

```bash
npm run dev
```

5. Open your browser and navigate to `http://localhost:5173` to see the application running.
